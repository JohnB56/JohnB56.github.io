Maturarbeit Passkeys

JavaScript single-page application secured with MSAL.js

Diese Webseite wurde erstellt, um die Implementierung einer Anmeldung mit Passkeys aufzuzeigen. Wenn man sich mit dem Knopf «Sign In» anmelden will, kann man als Option einen Passkey verwenden. Um sich anmelden zu können, muss man jedoch vom Webseiten Host eingeladen werden. Das Login mit einem Passkey ist sicherer, einfacher und schneller als die Authentifizierung mit einem Passwort.
